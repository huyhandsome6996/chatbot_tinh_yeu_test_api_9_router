# 💕 Tình Yêu AI - Chatbot Tư Vấn Tình Yêu

Chatbot tư vấn tình yêu sử dụng **9Router API** — lắng nghe và chia sẻ mọi tâm sự tình cảm.

## 🚀 Chạy trên máy

### Yêu cầu
- Node.js 18+
- 9Router đang chạy (localhost:20128)

### Cài đặt

```bash
git clone https://github.com/huyhandsome6996/chatbot_tinh_yeu_test_api_9_router.git
cd chatbot_tinh_yeu_test_api_9_router
npm install
npm run dev
```

Mở: **http://localhost:3000**

## 🔧 Cấu hình

Nhấp icon ⚙️ trên giao diện để cài đặt:

| Cài đặt | Mặc định | Mô tả |
|---------|----------|-------|
| Base URL | `http://localhost:20128` | Địa chỉ server 9Router |
| API Key | (từ 9Router Dashboard) | API key từ 9Router |
| Model | `gpt-4o` | Tên model AI |
